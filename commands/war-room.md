Convene the **war-room** on this decision:

$ARGUMENTS

Run the `war-room` skill. Frame the war, then **muster the council**: offer me the roster choice — a tailored roster (a light, question-scoped recon that reads only the slices this decision turns on and composes the seats for it) or the base five — before convening. Convene the chosen officers as non-colluding subagents (by default the Scout, the Strategist, the Marshal, the Quartermaster, and — last, against the emerging consensus — the Tenth Man), assemble the dossier (`STRATEGY` / `BATTLE-PLAN` / `BASELINE` / `HOW-WE-LOSE`), validate its structure with `hooks/validate-dossier.sh`, and present it to me as the Commander to seal.

If a named oracle (a test, a spec, an invariant) could settle this instead, say so and stop — that is a verification, not a war.
